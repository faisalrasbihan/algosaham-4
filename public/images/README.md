# Step Images for "Bagaimana Algosaham.ai Bekerja" Section

## Required Images

Add the following images to this directory for the "How It Works" section:

- `step-01.jpg` - For step 1: "Bangun Strategi Anda"
- `step-02.jpg` - For step 2: "Uji dengan Data Historis"
- `step-03.jpg` - For step 3: "Analisis Hasil Secara Mendalam"
- `step-04.jpg` - For step 4: "Terapkan Strategi Anda"

## Image Specifications

### Recommended:
- **Format**: JPG, PNG, or WebP
- **Size**: Square format (1:1 aspect ratio)
- **Dimensions**: At least 400x400px (will be displayed at 96x96px, but higher resolution ensures quality)
- **File Size**: Keep under 200KB per image for optimal performance
- **Style**: Images should be relevant to each step's theme and work well as backgrounds

### Alternative Formats:
- `step-01.png`, `step-02.png`, etc. (PNG format)
- `step-01.webp`, `step-02.webp`, etc. (WebP format for better compression)

## Image Themes (Suggestions)

1. **Step 01 - Bangun Strategi Anda**: Building/construction theme, tools, or strategy visualization
2. **Step 02 - Uji dengan Data Historis**: Charts, graphs, historical data visualization
3. **Step 03 - Analisis Hasil Secara Mendalam**: Analytics dashboard, metrics, or analysis tools
4. **Step 04 - Terapkan Strategi Anda**: Rocket, success, implementation, or execution theme

## Fallback

If images are not provided, the component will display a subtle gradient background with the number, so the section will still look good even without images.

## Usage

Once you add the images to this directory, they will automatically appear as backgrounds for the numbered steps (01-04) in the "Bagaimana Algosaham.ai Bekerja" section.
